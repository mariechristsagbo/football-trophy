# Football Trophy 


## Instructions 

1. Installez l'extension "Live Server" dans VS Code
2. Clic droit sur `index.html`
3. Sélectionnez "Open with Live Server"


## Fonctionnalités

- **Objectif du jeu** : Déplacez le gardien de but avec les flèches gauche/droite pour intercepter les balles qui descendent.

- **Touches de contrôle** :
  - **Flèche gauche** : Déplacer le gardien vers la colonne de gauche
  - **Flèche droite** : Déplacer le gardien vers la colonne de droite

- **Comment ça fonctionne alors ?**
  - Les balles descendent dans une des 3 colonnes
  - Quand une balle touche le gardien, vous marquez 1 point et la balle sort des colonnes de jeu
  - Si une balle sort de l'écran sans être interceptée, c'est **Game Over** !

### Touches de contrôle

- **Jouer** : Démarrer une nouvelle partie
- **Pause** : Mettre le jeu en pause
- **Play** : Reprendre le jeu après une pause
- **Meilleur score** : Afficher votre meilleur score enregistré


## Technologies

- **HTML5** : Structure sémantique
- **CSS3** : Styles, etc.
- **JavaScript** : Logique du jeu, gestion d'état, interactions
